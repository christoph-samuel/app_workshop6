import { NewsShort } from './news-short';

## React Component for rendering text

A basic div that renders some text

### Component usage
```js
<NewsShort heading="Eine Heading" text="Nachrichten Text" />
```

### Using props to customize the text

Modify the text to see it change live:
```js live
<NewsShort
heading="Einheitliche Tiebreak-Regel bei Grand-Slam-Turnieren"
text="Die Veranstalter der vier großen Tennis-Grand-Slam-Turniere – Australian Open in Melbourne, French Open in Paris, Wimbledon in London und US Open in New York – haben sich auf eine einheitliche Tiebreak-Regel geeinigt."
/>
```
